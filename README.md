# VIDEO GEN~ 

STEP 1 :
`git clone`

STEP 2 :
`npm i`

OPTIONAL: (NOT NEEDED FOR DEFAULT)

RUN `node main` TO FETCH DATA
RUN `srt.py` TO GENERATE CAPTIONS BASED ON THE AUDIO CREATE FROM MAIN

ADD ENV KEYS FOR ELVEN LABS AND PEXELS

STEP 3: FOR PREVIEW
`npm run start`

STEP 4 : TO RUN BUILD
`npm run build`
